# 启动模式设计方案

## 按键规划

| 引脚 | 功能 |
|------|------|
| GPIO33 | 开关机键（RTC 唤醒，深睡眠，未接，TODO） |
| SA (GPIO36) | 模式修饰键 |
| SD (GPIO38) | 模式修饰键 |

## 交互方式

### 开关机

| 操作 | 功能 |
|------|------|
| 短按 GPIO33 | 开机 → 默认纯射频 (CRSF) |
| 长按 GPIO33（2s） | 关机 → 保存 NVS → `esp_deep_sleep_start()` |

### 切换模式（运行时）

| 操作 | 功能 |
|------|------|
| 长按 GPIO33 + SA（2s 组合稳定） | 重启进入 **USB HID** |
| 长按 GPIO33 + SD（2s 组合稳定） | 重启进入 **BLE** |
| 长按 GPIO33 + SA + SD（2s 组合稳定） | 重启进入 **WiFi** |

切换流程：
1. 用户长按组合键 2 秒（期间引脚电平不变）
2. 检测到稳定组合 → `request_nvs_save()` 保存当前配置
3. 将目标模式写入 NVS 标记 (`boot_mode`)
4. `esp_restart()`
5. 重启后 `app_main()` 读取 `boot_mode` 标记 → 进入对应模式 → 清除标记
6. 无标记 → 默认纯射频 (CRSF)

### 待机唤醒（GPIO33 接入后）

| 操作 | 功能 |
|------|------|
| 短按 GPIO33 | `esp_deep_sleep_start()` 前已配置 RTC 唤醒 |
| RTC GPIO 低电平 → 芯片重启 → 默认进纯射频 |

### 组合键检测逻辑（主循环中）

```c
#define BOOT_MODE_NONE    0  // 无标记，默认纯射频
#define BOOT_MODE_CRSF    1
#define BOOT_MODE_USB     2
#define BOOT_MODE_BLE     3
#define BOOT_MODE_WIFI    4

#define HOLD_MS           2000  // 2s 长按检测

static uint8_t s_boot_mode = BOOT_MODE_NONE;  // NVS 持久化

// 主循环中每秒检测一次
static void check_mode_switch(gpio_num_t gpio33) {
    if (!gpio_get_level(gpio33)) {  // GPIO33 被按下
        bool sa = !gpio_get_level(RC_SWITCH_SA_PIN);
        bool sd = !gpio_get_level(RC_SWITCH_SD_PIN);
        
        uint32_t start = xTaskGetTickCount() * portTICK_PERIOD_MS;
        vTaskDelay(pdMS_TO_TICKS(HOLD_MS));
        
        // 2s 后再次读取，确认组合没变
        bool sa2 = !gpio_get_level(RC_SWITCH_SA_PIN);
        bool sd2 = !gpio_get_level(RC_SWITCH_SD_PIN);
        bool p33 = !gpio_get_level(gpio33);
        
        if (!p33) return;  // 松手了，忽略
        
        uint8_t mode = BOOT_MODE_NONE;
        if (!sa2 && !sd2)      mode = BOOT_MODE_WIFI;
        else if (!sa2)          mode = BOOT_MODE_USB;
        else if (!sd2)          mode = BOOT_MODE_BLE;
        
        if (mode != BOOT_MODE_NONE && sa == sa2 && sd == sd2) {
            // 组合稳定 → 保存标记 → 重启
            request_nvs_save();
            nvs_set_u8("storage", "boot_mode", mode);
            esp_restart();
        }
    }
}
```

## 启动流程（app_main）

```
NVS 初始化 → 加载配置 → 读 boot_mode 标记
    │
    ├── BOOT_MODE_USB  → usb_init()          → 清除标记
    ├── BOOT_MODE_BLE  → ble_init()          → 清除标记
    ├── BOOT_MODE_WIFI → rc_wifi_server_init() → 清除标记
    └── 默认/BOOT_MODE_CRSF → 纯射频，无初始化
```

## 实现计划

- **第一阶段**（当前）：默认纯射频，SA+SD 组合切换 WiFi
  - GPIO33 未接入时，先用 SA+SD 做模式切换（无长按检测，直接 NVS 标记 + 重启）
  - 用完切回 CRSF 模式
  
- **第二阶段**（GPIO33 接入后）：
  - GPIO33 短按开机（RTC 唤醒）
  - GPIO33 长按 2s 关机（深睡眠）
  - GPIO33 + SA/SD 组合长按 2s 切换模式 + 重启
