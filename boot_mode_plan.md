# 开机模式重构方案（已归档）

> ⚠️ 本方案已被当前实现取代，详见 `docs/启动模式设计.md`

## 原先思路
- USB 应默认开启，插线即用，无需开关机切换模式
- BLE 不应默认启动，改为按需开关（NVS 记忆）
- WiFi 是配置/调试用途，应作为特殊模式而非常规模式
- 全开 USB HID 会导致无法使用 USB 串口烧录/调试，需一个串口调试模式

## 当前方案（已实现）

见 `docs/启动模式设计.md`。

核心区别：
- 改为 SA 按键长按 + 右摇杆方向选择，不再依赖 GPIO33
- USB MSC（U盘模式）作为可选模式加入
- 默认模式改为 WiFi AP
